Graphing Quadratic Functions

The graph of a quadratic function is a parabola. It can open upward or downward depending on the value of a in the equation f(x) = ax² + bx + c.

Key Features of a Parabola

  • Vertex: The highest or lowest point of the parabola
  • Axis of Symmetry: A vertical line that passes through the vertex, written as x = h
  • Direction: Opens upward if a > 0, downward if a < 0
  • Y-intercept: The point where the graph crosses the y-axis (when x = 0)
  • X-intercepts (roots): The points where the graph crosses the x-axis

Using Vertex Form to Graph

The vertex form of a quadratic is:

f(x) = a(x - h)² + k

  • (h, k) is the vertex
  • Start by plotting the vertex
  • Use a to determine how wide or narrow the parabola is
  • Plot points symmetrically around the vertex

Example:

Graph f(x) = 1(x - 2)² + 3

  • Vertex is at (2, 3)
  • Parabola opens upward since a = 1
  • Plot symmetric points like (1, 4) and (3, 4)

Interactive Example: View on Desmos

Using a Table of Values

Pick values of x around the vertex and plug them into the equation to find y. Then plot these points to sketch the graph.

Effect of the Coefficient "a"

  • If |a| > 1: the graph is narrower
  • If 0 < |a| < 1: the graph is wider
  • If a < 0: the graph opens downward

Interactive Example: Try different values of "a" on Desmos

Using the Standard Form

If you start with f(x) = ax² + bx + c:

  • Find the vertex using the formula: x = -b / 2a
  • Plug that x-value into the equation to get y and find the vertex
  • Plot the vertex and symmetric points

Interactive Example: Graph a standard form equation on Desmos

Conclusion

Quadratic graphs show important information like maximum or minimum points and where the function equals zero. Learning how to sketch and analyze these graphs is a key part of Pre-Calculus 11.
