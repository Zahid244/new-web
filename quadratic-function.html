What is a Quadratic Function?

A quadratic function is a type of function that creates a U-shaped curve called a parabola. The standard form is:

f(x) = ax² + bx + c

Where a, b, and c are real numbers, and a ≠ 0.

Forms of Quadratic Functions

  • Standard Form: f(x) = ax² + bx + c
  • Vertex Form: f(x) = a(x - h)² + k
  • Factored Form: f(x) = a(x - r₁)(x - r₂)

Parts of a Parabola

  • Vertex: the highest or lowest point on the graph
  • Axis of Symmetry: the vertical line through the vertex
  • Direction: opens up if a > 0, opens down if a < 0
  • Y-intercept: the value of f(x) when x = 0

Example:

Let’s take this quadratic function:

f(x) = 2(x - 3)² + 1

  • This is in vertex form: a = 2, h = 3, k = 1
  • The vertex is at (3, 1)
  • The parabola opens upward because a = 2 > 0

If you expanded this equation, you'd get the standard form.

Why It Matters

Quadratic functions appear in real life when modeling things like projectile motion, area, and optimization problems. They are an important part of algebra and pre-calculus.
