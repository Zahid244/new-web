What is a Quadratic Equation?

A quadratic equation is a second-degree polynomial equation in one variable, which means the highest exponent of the variable is 2. The general (standard) form is:

ax² + bx + c = 0

Where:

  • a, b, and c are real numbers
  • a ≠ 0 (If a = 0, it becomes a linear equation)

Methods of Solving Quadratic Equations

1. Factoring

Write the equation in the form ax² + bx + c = 0 and factor the left-hand side.

Example:

x² - 5x + 6 = 0

Factor: (x - 2)(x - 3) = 0

Set each factor to zero: x = 2 or x = 3

2. Using the Quadratic Formula

For any equation in standard form, use the formula:

x = (-b ± √(b² - 4ac)) / (2a)

Steps:

  • Identify a, b, and c from the equation
  • Calculate the discriminant: D = b² - 4ac
  • If D > 0, you get two real solutions
  • If D = 0, one real solution
  • If D < 0, two complex (imaginary) solutions

Example:

2x² + 3x - 2 = 0

Here, a = 2, b = 3, c = -2

Discriminant: D = 3² - 4(2)(-2) = 9 + 16 = 25

x = (-3 ± √25) / 4 → x = (-3 ± 5) / 4

Solutions: x = 0.5 and x = -2

3. Completing the Square

This method rewrites the quadratic into a perfect square trinomial. Use when factoring doesn't work easily.

Steps:

  1. Move the constant to the other side
  2. Divide all terms so the coefficient of x² is 1
  3. Add the square of half of b to both sides
  4. Factor and solve

Example:

x² + 6x = 7

Half of 6 is 3 → 3² = 9

Add 9 to both sides: x² + 6x + 9 = 16

Now: (x + 3)² = 16

Take the square root: x + 3 = ±4

Solutions: x = 1 and x = -7

Other Notes

Discriminant Summary:

  • D = b² - 4ac
  • If D > 0: Two real and distinct roots
  • If D = 0: One real root (a repeated solution)
  • If D < 0: Two complex roots (no real solutions)

Special Cases:

  • Perfect square trinomial: x² + 6x + 9 = (x + 3)²
  • Difference of squares: x² - 9 = (x - 3)(x + 3)

Conclusion

Understanding how to solve quadratic equations is a key skill in algebra and will help in calculus, physics, engineering, and many real-life problems. Master the three main solving methods and know when to use each!
